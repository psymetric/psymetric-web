# VS Code Page Command Center

## Purpose

Defines the Page Command Center feature of the VEDA VS Code extension.

This feature provides context-aware SEO intelligence for the page currently open in the editor.

## Concept

The Page Command Center combines:

- Content Graph context
- SERP Observatory signals
- Strategy reasoning

This allows VEDA to analyze a page within the context of the live search ecosystem.

## Context Sources

### Content Graph

Provides:

- page identity
- page type
- internal links
- anchor text
- schema usage
- entity/topic coverage

### SERP Observatory

Provides:

- volatility
- SERP regime
- AI overview presence
- domain dominance
- SERP similarity

### Strategy Layer

Provides reasoning and suggested actions.

## Example Output

VEDA may report:

- page archetype mismatch
- missing support pages
- weak internal linking
- schema gaps

## Actions

The extension may expose commands such as:

- Investigate current page
- Compare page vs SERP leaders
- Generate execution plan
- Suggest internal links

## Future

Later versions may include:

- territory intelligence
