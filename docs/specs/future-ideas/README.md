# Future Ideas (Speculative)

This directory contains experimental concepts and long‑range research ideas for VEDA.

These are **not active development targets**.

## Rules

1. Ideas in this folder must **not influence current implementation decisions**.
2. No production code should reference these documents.
3. Promotion of an idea out of this folder requires an explicit architectural decision.
4. Any promoted idea must have hammer coverage before entering active development.

Think of this directory as the **research notebook for VEDA**. It allows exploration of ambitious concepts without introducing architectural drift into the core system.

## Examples of ideas that may appear here

- Strategy synthesis layer
- Global SERP anomaly detection
- Algorithm update early warning system
- External research signal monitoring (LLM papers, patents)

These documents exist to capture thinking early while keeping the core system disciplined.
